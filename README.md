# 📊 Retail Sales Dashboard (Excel Project)

## 📌 Overview

This project is an interactive **Retail Sales Dashboard** built using Microsoft Excel.
It provides insights into sales performance, customer behavior, and product trends.

---

## 🎯 Objectives

* Analyze total revenue, profit, and orders
* Identify top-performing products and categories
* Understand customer demographics (age, gender)
* Compare sales across different countries

---

## 📈 Key Insights

* 🚴‍♂️ **Bikes** generate the highest revenue among all categories
* 🌍 **United States** is the top-performing country
* 👨‍👩‍👧 **Adults** contribute the most to overall revenue
* 🏆 Top 10 products drive a significant portion of total sales

---

## 🛠️ Tools & Techniques Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers (for interactivity)
* KPI Cards
* Data Cleaning & Formatting
